This is a workout program generator. The user creates a weekly schedule, sets some parameters, and the application generates a workout program based on the values entered by the user.

Features:

  • Database management with SQLAlchemy
  • User authentication with Flask-Login
  • Form validation with WTForms
  • Dynamic HTML with Jinja
  • Responsive layout with Flask-Bootstrap and CSS media queries

Info:

  • Language: Python
  • Framework: Flask
  • Main libraries: SQLAlchemy, Flask-Login, WTForms, Flask-Bootstrap
